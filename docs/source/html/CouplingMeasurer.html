CouplingMeasurer.py¶

This module provides a class for measuring the coupling of a transformer. It is compliant with the pymeasure Instrument class.

class OST.CouplingMeasurer.CouplingMeasurer(adapter, name='Coupling Measurer', **kwargs)[source]¶

Bases: Instrument

A class to interface with a coupling measurement instrument. This class provides methods to control the instrument and retrieve measurement data. It allows setting the frequency and voltage for measurements, and retrieving various coupling parameters.
